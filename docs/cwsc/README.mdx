# Introduction

`cwsc` is the compiler for CWScript.

```sh
$ cwsc --help

VERSION
  @terran-one/cwsc/0.1.0 darwin-arm64 node-v20.0.0

USAGE
  $ cwsc [COMMAND]

COMMANDS
  autocomplete  display autocomplete installation instructions
  help          Display help for cwsc.
  parse         Parse a CWScript source file into AST.
```
